## 环境配置

按照操作系统类型安装python（官方使用的环境为3.7.6），安装方式各异。安装成功后在cmd/bash中定位到该文件夹，并使用如下命令安装依赖。

```text
pip install -r requirements.txt

如果使用的是conda环境
 conda install --yes --file requirements.txt
（建议使用conda环境安装，否则会安装很多不必要的库）
```

mac/linux用户需将pip切换成pip3
